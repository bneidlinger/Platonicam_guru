Project: Enterprise Surveillance Design Assistant

Goal: A local-first RAG (Retrieval-Augmented Generation) system to parse vendor PDFs (Hanwha, Axis, Bosch) and provide design-level intelligence for Physical Security Systems Engineering.

Architecture Overview

[PDF Library] -> [PyMuPDF Parser] -> [Recursive Character Splitting]
-> [Ollama Embedding (Nomic-Text)] -> [ChromaDB Vector Store]
-> [User Query] -> [Llama 4.1 Scout Context Retrieval] -> [Web UI Response]

Key Capabilities

  • Automated POE Budgeting: Scrape "Max Power Consumption" from tables and sum per project.
  • Accessory Logic: Identify matching mounts/pendants by cross-referencing model numbers in text.
  • Config Retrieval: Instant access to default IPs, credentials, and RTSP string formats.
